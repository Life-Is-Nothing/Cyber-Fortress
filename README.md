# 🏰 Cyber-Fortress

> Custom Linux Security Monitoring & Hardening Script — by [@ibramoha2](https://github.com/ibramoha2)

## 🔧 Ce que ça vérifie

| Module | Vérifications |
|--------|--------------|
| 🔒 Services | UFW, fail2ban, CrowdSec, auditd, AppArmor |
| 🔑 SSH | PermitRootLogin, PasswordAuth, Port |
| 🌐 Réseau | Ports ouverts, IPv6 |
| 📦 Updates | Unattended-upgrades |
| 📋 Logs | Tentatives SSH échouées, IPs bannies |

## 🚀 Usage

```bash
git clone https://github.com/ibramoha2/Cyber-Fortress
cd Cyber-Fortress
chmod +x cyber_fortress.sh
sudo ./cyber_fortress.sh
```
